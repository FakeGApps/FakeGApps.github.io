Frequently Asked Questions

1. Do i need to clean flash for every new FakeGApps release ?

No, You don’t have to clean install, whenever a new FakeGApps update come. You can simply dirty install over last FakeGApps installation.

2. Do i need to re-install FakeGApps EVERY time, I update my ROM ?

Probably not. If your ROM supports addon.d backup functionality, it’s not necessary to install FakeGApps with each ROM update, as long as you don’t manually wipe the System partition. The backup script will backup your installed GApps and restores them after your ROM has been updated. However, it is a good idea to install an updated GApps from time to time as functionality may be added or improvements made.

3. I am getting connection error with Google Playstore. How to fix this ?

Go to Settings > Apps > Google Play Store > Mobile data & WiFi > toggle Allow Network Access.

4. How to Create/Edit configuation file and where to place this before installation ?

You can Create/Edit configuation file in Android or Linux machine, for Windows OS Text-File-Format must be UNIX/OSX. Notepad++ can be used to Create/Edit. It is available for both Windows and Linux OS. On Android, you can have text editor of your choice. This file must be placed in your device internal storage before installation.

5. LineageOS Recovery does not support data decryption. How to use configuation file with it ?

With LineageOS Recovery, Adding configuation file to internal storage will not work. Because of encrypted data partition. You need to re-pack GApps package with file. The file must goes in root of un-packed GApps package else it won’t work.

6. I have NLP Version installed, Can i switch to other versions without clean flash ?

Yes, You can shuffle between different versions without clean flash.

7. Do i need to perform additional steps after installation of BSP or GPS version ?

Yes, You need to turn on Google Device Registration & Cloud Messaging from microG Settings.

8. Do i need to flash every new FakeGApps release on top of last installation ?

No, You certainly don't have to, if everything is fine with your last installation, Just keep this as is and switch to new FakeGApps release, When you clean flash ROM again or want to use a feature which is available in new release and not in last. However you can keep the latest version downloaded.

9. FakeGApps installation stucked at "Mounting /apex" How to fix this ?

Use some different RECOVERY.

10. How to install FakeGApps and What is the use case of Advance Features ?

Install from Recovery like Lineage Recovery, TWRP, etc. Methods under Advance Features are Optional.

11. What are the different methods of flashing a FakeGApps Package ?

We offer installation from Recovery like TWRP, Lineage Recovery, etc and installation from Magisk or KernelSU Application. That's too from JUST one FakeGApps Package. Furthermore installation can be done in two ways from Magisk or KernelSU Application.

12. What is Altair and How we can make use of it ?

A Companion App for FakeGApps. It can create configuration file with selected options and rebuild FakeGApps Package with that configuration.

13. Where can i find FakeGApps Package rebuilded with Altair ?

Rebuilded FakeGApps Package can be found at Downloads/Altair folder. File has REPACK text at the end.

14. What is the use of "Module Installation" listed in Altair ?

Please check Advance Features and Options to learn more about it.
